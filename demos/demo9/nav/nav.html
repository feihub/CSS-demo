导航栏基本上就是链接列表，因此我们通常使用ul和li元素来创建导航栏

以下是一个最简单的导航栏例子:

  • HOME
  • HTML
  • CSS
  • Javascript
  • Vue
  • React
  • Angular

在实际开发中, 根据系统UI设计,导航栏一般有垂直导航栏和水平导航栏两种,且将单调的HTML菜单转换为美观的导航栏



Code: 
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>导航栏</title>
        <style>
            ul {
                /*  清除项目符号, 导航条不需要列表项标记 */
                list-style-type: none;
                /*  清除浏览器的默认设置 */
                margin: 0;
                padding: 0;
            }
        </style>
    </head>
    <body>
        <div class="default_nav">
            <h3>导航栏基本上就是链接列表，因此我们通常使用ul和li元素来创建导航栏</h3>
            <p>以下是一个最简单的导航栏例子:</p>
            <div class="div_line">
                <ul>
                    <li><a href="">HOME</a></li>
                    <li><a href="">HTML</a></li>
                    <li><a href="">CSS</a></li>
                    <li><a href="">Javascript</a></li>
                    <li><a href="">Vue</a></li>
                    <li><a href="">React</a></li>
                    <li><a href="">Angular</a></li>
                </ul>
            </div>
        </div>
    </body>
    </html>
