1. 什么是CSS

    • CSS是一种标记性语言，用来描述如何显示HTML元素的
    • 可以实现页面结构和表现的分离，HTML控制网页结构，CSS控制样式

  2. CSS语法

    • 选择器 {属性: 属性值; 属性: 属性值...}
      • eg: h1 {color: red; font-color: 14px;} (请勿在属性值和单位之间添加空格[例如: font-size: 14 px;])

  3. CSS选择器

    简单选择器（根据名称、id、类来选取元素  *id 名称不能以数字开头

    1. CSS 元素选择器
    2. CSS id 选择器
    3. CSS 类选择器
    4. CSS 通用选择器
    5. CSS 分组选择器(h1、h2、p元素具有相同的样式定义，最好对选择器进行分组， 以最大程度的缩减代码。选择器分组用逗号来分隔)

    组合器选择器（根据它们之间的特定关系来选取元素）

    1. 后代选择器 (空格)
    2. 子选择器 (>)
    3. 相邻兄弟选择器 (+)
    4. 通用兄弟选择器 (~), A~B选择A元素之后所有同层级B元素

    伪类选择器（根据特定状态选取元素）

    主要是用于定义元素的特殊状态：

    • 设置鼠标悬停在元素上时的样式
    • 为已访问和未访问的链接设置不同样式
    • 设置元素获得焦点是的样式

    伪类的语法： selector:: pseudo-element { property: value }


    • 锚伪类（链接能够以不同的方式显示）+
    • 伪类和CSS类结合使用
    • 简单的工具提示悬停
    • CSS- :first-child伪类（伪类与指定元素匹配：该元素是另一个元素的第一个子元素）
      • 选择器匹配作为任何元素的第一个子元素的任何< p >元素
      • 匹配所有< p > 元素中的首个 < i > 元素
      • 匹配所有首个< p > 元素中的所有 < i > 元素
        • CSS-lang伪类，伪类允许为不同的语言定义特殊的规则。:lang为属性为lang="en"的< p >元素定义引号
