1. 使用 padding 属性将 div 元素垂直居中

元素垂直居中

2. 使用flex将 div 元素垂直和水平居中

垂直和水平居中

3. 使用position和transform将 div 元素垂直和水平居中

垂直和水平居中

Code: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>垂直对齐块元素</title>
    <style>
        .div_line {
            width: 45%;
            float: left;
            background-color: blanchedalmond;
        }
        .block_1 {
            padding: 60px 0;
            border: 3px solid black;
        }
        .block_2 {
            /* 同时将 justify-content 和 align-items 属性设置为center，将使其在中心（在两个轴上）对齐。 */
            display: flex;
            height: 100px;
            justify-content: center;
            align-items: center;
            border: 3px solid black;
        }
        .block_3 {
            height: 100px;
            position: relative;
            border: 3px solid black;
        }
        .content {
            margin: 0px;
            position: absolute;
            top: 50%;
            left: 50%;
            -ms-transform: translate(-50%, -50%);
            transform: translate(-50%, -50%); 
        }
    </style>
</head>
<body>
    <div class="div_line">
        <h1>1. 使用 padding 属性将 div 元素垂直居中</h1>
        <div class="block_1">
            <p>元素垂直居中</p>
        </div>
    
        <h1>2. 使用flex将 div 元素垂直和水平居中</h1>
        <div class="block_2">
            <p>垂直和水平居中</p>
        </div>
    
        <h1>3. 使用position和transform将 div 元素垂直和水平居中</h1>
        <div class="block_3">
            <p class="content">垂直和水平居中</p>
        </div>
    </div>

    <div class="div_line" style="background-color: #fff;">
        Code:
    </div>
</body>
</html>
