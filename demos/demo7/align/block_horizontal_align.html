1. 水平居中对齐块元素:

要使块元素（例如div）水平居中，可以使用 margin: auto。设置元素的宽度防止其延伸到容器的边缘。然后，元素将占用指定的宽度，剩余空间将在两个外边距之间平均分配：

块元素居中

2. 水平居中图像:

要使图像居中，需将左右外边距设置为auto，并使其成为一个块元素。

3. 水平左右对齐:

1. 使用position: absolute;

块元素向右对齐



2. 使用float

块元素向右对齐

    <style>
        .center {
            margin: auto;
            /* 如果未设置宽度或者宽度为100%，则居中对齐无效 */
            width: 50%;
            border: 5px solid black;
            padding: 10px;
        }
        
        img {
            /* img本身是行内元素 如果这里未设置为块元素，则居中对齐无效 */
            display: block;
            margin-left: auto;
            margin-right: auto;
        }
        .right {
            position: absolute;
            right: 0px;

        }

    </style>
