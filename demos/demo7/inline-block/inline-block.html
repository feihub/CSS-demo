使用inline-block创建水平导航链接

默认地，列表项是垂直显示的，如下：

  • NCS
  • Making
  • IT
  • Happen

而在本例中，我们使用 display: inline-block 来水平地显示它们（并排）。

  • NCS
  • Making
  • IT
  • Happen
Code: 
                <!DOCTYPE html>
                <html lang="en">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>inline-block</title>
                    <style>
                        .div-line {
                            width: 45%;
                            float: left;
                        }
                        .default {
                            background-color: #a0a0b1;
                        }
                        .nav {
                            background-color: #a0a0b1;
                            text-align: center;
                            list-style-type: none;
                        }
                        .nav li {
                            /* 使用inline-block来让每个列表项水平排列 */
                            display: inline-block;
                            font-size: 20px;
                            padding: 30px;
                        }
                    </style>
                </head>
                <body>
                    <div class="div-line">
                        <h3>使用inline-block创建水平导航链接</h3>
                        <p>默认地，列表项是垂直显示的，如下：</p> 
                        <ul class="default">
                            <li><a href="https://www.ncs.com.sg/public/ncs/en.html" target="_blank">NCS</a></li>
                            <li><a href="https://www.ncs.com.sg/public/ncs/en.html" target="_blank">Making</a></li>
                            <li><a href="https://www.ncs.com.sg/public/ncs/en.html" target="_blank">IT</a></li>
                            <li><a href="https://www.ncs.com.sg/public/ncs/en.html" target="_blank">Happen</a></li>
                        </ul>
                        <p>而在本例中，我们使用 display: inline-block 来水平地显示它们（并排）。</p>
                        <ul class="nav">
                            <li><a href="https://www.ncs.com.sg/public/ncs/en.html" target="_blank">NCS</a></li>
                            <li><a href="https://www.ncs.com.sg/public/ncs/en.html" target="_blank">Making</a></li>
                            <li><a href="https://www.ncs.com.sg/public/ncs/en.html" target="_blank">IT</a></li>
                            <li><a href="https://www.ncs.com.sg/public/ncs/en.html" target="_blank">Happen</a></li>
                        </ul>
                    </div>
                </body>
                </html>
