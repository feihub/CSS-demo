相邻兄弟选择器(+)

相邻兄弟选择器匹配所有作为指定元素的相邻同级的元素。兄弟（同级）元素必须具有相同的父元素，“相邻”的意思是“紧随其后”。

下面的例子仅选择紧随div元素之后的p元素：

div中的p标签1

div中的p标签2

p标签3，不在div中

p标签4，不在div中

Code： 
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>相邻兄弟选择器</title>
        <style>
            /* 仅选择紧随div之后的p元素w */
            div + p {
                background-color: peru;
            }
        </style>
    </head>
    <body>
        <h1>相邻兄弟选择器(+)</h1>
        <p>相邻兄弟选择器匹配所有作为指定元素的相邻同级的元素。兄弟（同级）元素必须具有相同的父元素，“相邻”的意思是<b>“紧随其后”</b>。
            下面的例子仅选择紧随div元素之后的所有p元素：</p>
        <div>
            <p>div中的p标签1</p>
            <p>div中的p标签2</p>
        </div>
        <p>p标签3，不在div中</p>

        <!-- p标签4的背景颜色不会改变，因为它没有紧随div之后 -->
        <p>p标签4，不在div中</p> 
    </body>
    </html>
