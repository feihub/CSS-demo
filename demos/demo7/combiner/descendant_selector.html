后代选择器

后代选择器匹配作为指定元素后代的所有元素。

div中的内容1

div中的内容2

不在div中的内容4

Code: 
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
        <style>
            /* p为div的后代元素 */
            div p{
                background-color: bisque;
            }
        </style>
    </head>
    <body>
        <h1>后代选择器</h1>
        <p>后代选择器匹配作为指定元素后代的所有元素。</p>
        <div>
            <p>div中的内容1</p>
            <p><span>div中的内容2</span></p>
        </div>
        <p>不在div中的内容4</p>
    </body>
    </html>
