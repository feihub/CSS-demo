子选择器

子选择器 (>) 选择属于指定元素子元素的所有元素。

div 中的内容1

div 中的内容2

div 中的内容3

内容4不在 div 中

Code: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        /* 子选择器 > */
        div > p {
            background-color: olivedrab;
        }
    </style>
</head>
<body>
    <h1>子选择器</h1>
    <p>子选择器 (>) 选择属于指定元素子元素的所有元素。</p>
    <div>
        <p>div 中的内容1</p>
        <p>div 中的内容2</p>
        <section><p>div 中的内容3</p></section> <!-- 非子但属后代 所以不会被选择-->
    </div>
    <p>内容4不在 div 中</p>
</body>
</html>
