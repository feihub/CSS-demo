通用兄弟选择器(~)

通用兄弟选择器匹配属于指定元素的同级元素的所有元素。

下面的例子选择属于div元素的同级元素的所有p元素：

div中的p标签1

div中的p标签2

p标签3，不在div中

p标签4，不在div中

Code: 
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>通用兄弟选择器</title>
        <style>
            /* 选择与div同级的所有p元素 */
            div ~ p {
                background-color: peru;
            }
        </style>
    </head>
    <body>
        <h1>通用兄弟选择器(~)</h1>
        <p>通用兄弟选择器匹配属于指定元素的同级元素的所有元素。</p>
        <p>下面的例子选择属于div元素的同级元素的所有p元素：</p>
        <div>
            <p>div中的p标签1</p>
            <p>div中的p标签2</p>
        </div>
        <p>p标签3，不在div中</p>
        <p>p标签4，不在div中</p> 
    </body>
    </html>
