1. 什么是CSS

    • CSS是一种标记性语言，用来描述如何显示HTML元素的
    • 可以实现页面结构和表现的分离，HTML控制网页结构，CSS控制样式

  2. CSS语法

    • 选择器 {属性: 属性值; 属性: 属性值...}
      • eg: h1 {color: red; font-color: 14px;} (请勿在属性值和单位之间添加空格[例如: font-size: 14 px;])

  3. CSS选择器

    简单选择器（根据名称、id、类来选取元素  *id 名称不能以数字开头

    1. CSS 元素选择器
    2. CSS id 选择器
    3. CSS 类选择器
    4. CSS 通用选择器
    5. CSS 分组选择器(h1、h2、p元素具有相同的样式定义，最好对选择器进行分组， 以最大程度的缩减代码。选择器分组用逗号来分隔)

    组合器选择器（根据它们之间的特定关系来选取元素）

    1. 后代选择器 (空格)
    2. 子选择器 (>)
    3. 相邻兄弟选择器 (+)
    4. 通用兄弟选择器 (~), A~B选择A元素之后所有同层级B元素

    伪类选择器（根据特定状态选取元素）

    主要是用于定义元素的特殊状态：

    • 设置鼠标悬停在元素上时的样式
    • 为已访问和未访问的链接设置不同样式
    • 设置元素获得焦点是的样式

    伪类的语法： selector:: pseudo-element { property: value }


    • 锚伪类(链接能够以不同的方式显示)
    • 简单的工具提示悬停
    • CSS- :first-child伪类（伪类与指定元素匹配：该元素是另一个元素的第一个子元素）
      • 选择器匹配作为任何元素的第一个子元素的任何< p >元素
      • 匹配所有< p > 元素中的首个 < i > 元素
      • 匹配所有首个< p > 元素中的所有 < i > 元素
        • CSS-lang伪类，伪类允许为不同的语言定义特殊的规则。:lang为属性为lang="en"的< p >元素定义引号

    伪元素选择器（选取元素的一部分并设置其样式）

    设置元素指定部分的样式：

    • 设置元素的首字母、首行的样式
    • 在元素的内容之前或之后插入内容

    伪类的语法： selector::pseudo-element { property: value }

    • ::first-line 伪元素（用于向文本的首行添加特殊样式）
    • ::first-letter 伪元素(用于向文本的首字母添加特殊样式)
    • CSS - ::before 伪元素(可用于在元素内容之前插入一些内容)
    • CSS - ::after 伪元素(可用于在元素内容之后插入一些内容)

  4. 属性选择器（根据属性或属性值来选取元素）

    • CSS [attribute] 选择器(用于选取带有指定属性的元素) 选择所有带有 target 属性的<a>元素
    • CSS [attribute="value"] 选择器(用于选取带有指定属性和值的元素) 选取所有带有 target="_blank" 属性的 <a> 元素
    • CSS [attribute~="value"] 选择器(选取属性值包含指定词的元素)选取 title 属性包含 "flower" 单词的所有元素
    • CSS [attribute|="value"] 选择器(选取指定属性以指定值开头的元素[值必须是完整或单独的单词，比如 class="top" 或者后跟连字符的，比如 class="top-text"。])选取 class 属性以 "top" 开头的所有元素
    • CSS [attribute^="value"] 选择器(选取指定属性以指定值开头的元素[值不必是完整单词]) 选取 class 属性以 "top" 开头的所有元素
    • CSS [attribute$="value"] 选择器(选取指定属性以指定值结尾的元素[值不必是完整单词]) 选取 class 属性以 "test" 结尾的所有元素
    • CSS [attribute*="value"] 选择器(选取属性值包含指定词的元素[值不必是完整单词]) 选取 class 属性包含 "te" 的所有元素
