CSS [attribute="value"] 选择器

target="_blank" 的链接会有黄色背景：

w3school.com.cn disney.com wikipedia.org
Code: 
                
            <html>
            <head>
            <style>
                a[target=_blank] {
                    background-color: yellow;
                }
            </style>
            </head>
            <body>
                <h1>CSS [attribute="value"] 选择器</h1>
                <p>target="_blank" 的链接会有黄色背景：</p>
                <a href="https://www.w3school.com.cn">w3school.com.cn</a><br>
                <a href="http://www.disney.com" target="_blank">disney.com</a><br>
                <a href="http://www.wikipedia.org" target="_top">wikipedia.org</a><br>
            </body>
            </html>
