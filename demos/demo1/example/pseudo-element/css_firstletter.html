您可以使用 ::first-letter 伪元素为文本的第一个字符添加特殊效果！

Code: 
                <html>
                <head>
                <style>
                    p::first-letter {
                        color: #ff0000;
                        font-size: xx-large;
                    }
                </style>
                </head>
                <body>
                    <p>您可以使用 ::first-letter 伪元素为文本的第一个字符添加特殊效果！</p>
                </body>
                </html>
