后代选择器

后代选择器匹配作为指定元素后代的所有元素。

div 中的段落 1。

div 中的段落 2。

div 中的段落 3。

段落 4。不在 div 中。

段落 5。不在 div 中。

---------------------------------
Code: 
        <html>
        <head>
            <style>
                div p {
                background-color: yellow;
                }
            </style>
        </head>
        <body>
            <h1>后代选择器</h1>
            <p>后代选择器匹配作为指定元素后代的所有元素。</p>
            <div>
                <p>div 中的段落 1。</p>
                <p>div 中的段落 2。</p>
                <section>
                    <p>div 中的段落 3。</p>
                </section>
            </div>
            <p>段落 4。不在 div 中。</p>
            <p>段落 5。不在 div 中。</p>
        </body>
        </html>
