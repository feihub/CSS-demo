相邻兄弟选择器

相邻的同胞选择器（+）选择所有作为指定元素的相邻的同级元素。

div 中的段落 1。

div 中的段落 2。

段落 3。不在 div 中。

段落 4。不在 div 中。

Code: 
            <html>
            <head>
                <style>
                    div + p {
                    background-color: yellow;
                    }
                </style>
            </head>
            <body>
                <h1>相邻兄弟选择器</h1>
                <p>相邻的同胞选择器（+）选择所有作为指定元素的相邻的同级元素。</p>
                <div>
                  <p>div 中的段落 1。</p>
                  <p>div 中的段落 2。</p>
                </div>
                <p>段落 3。不在 div 中。</p>
                <p>段落 4。不在 div 中。</p>
            </body>
            </html>
