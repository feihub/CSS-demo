页眉

页眉（header）通常位于网站顶部（或顶部导航菜单的正下方）。它通常包含徽标（logo）或网站名称

页脚

页脚位于页面底部。它通常包含诸如版权和联系方式之类的信息

Code: 

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>响应式布局</title>
            <style>
                * {
                    box-sizing: border-box;
                }
                body {
                    margin: 0;
                }
                /* 设置页眉样式 */
                .header {
                    background-color: #f1f1f1;
                    text-align: center;
                    padding: 12px;
                }
                /* 设置页脚样式 */
                .footer {
                    background-color: #eee;
                    text-align: center;
                    padding: 8px;
                }
                /* 设置顶部导航栏样式 */
                .topnav {
                    overflow: hidden;
                    background-color: #333;
                    text-align: center;
                }
                .topnav a{
                    float: left;
                    display: block;
                    color: white;
                    text-decoration: none;
                    padding: 18px 16px;
                }
                .topnav a:hover {
                    color: black;
                    background-color: #ddd;
                }
                /* 将三个内容列并排 */
                .column {
                    float: left;
                    padding: 12px;
                    text-align: center;
                }
                /* 设置左，右内容宽度为25% */
                .side {
                    width: 25%;
                }
                /* 设置主内容宽度为50% */
                .main {
                    width: 50%;
                }
                .row:after {
                    /* 清除列之后的浮动 */
                    clear: both;
                    display: table;
                    content: "";
                }
                /* 响应式布局 - 创建堆叠而非并排的三列 */
                @media screen and (max-width: 600px) {
                    .column.side, .column.middle {
                        width: 100%;
                    }
                }
            </style>
        </head>
