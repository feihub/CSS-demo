CSS文本转换

text-transform属性用于设置文本中文字的大小写

它可用于将所有内容转换为大写或小写字母，或将每个单词的首字母大写：

Hello

WorLD

ncs Making IT Happen

Code:

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>CSS文本转换</title>
        <style>
            /* 所有文本内容转换为大写 */
            .uppercase {
                text-transform: uppercase;
            }
            /* 所有文本内容转换为小写 */
            .lowercase {
                text-transform: lowercase;
            }
            /* 所有文本首字母转为大写 */
            .capitalize {
                text-transform: capitalize;
            }
        </style>
    </head>
    <body>
        <h1>CSS文本转换</h1>
        <p><b>text-transform</b>属性用于设置文本中文字的大小写</p>
        <p>它可用于将所有内容转换为大写或小写字母，或将每个单词的首字母大写：</p>
        <p class="uppercase">Hello</p>
        <p class="lowercase">WorLD</p>
        <p class="capitalize">ncs Making IT Happen</p>
    </body>
    </html>
