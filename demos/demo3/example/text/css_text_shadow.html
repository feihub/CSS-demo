CSS文本阴影

text-shadow 属性为文本添加阴影

text-shadow: 垂直阴影 水平阴影 模糊效果 颜色

文本阴影效果

Code: 

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>CSS文本阴影</title>
        <style>
            h2 {
                /* text-shadow: 垂直阴影 水平阴影 模糊效果 颜色 */
                text-shadow: 3px 4px 5px orange;
            }
        </style>
    </head>
    <body>
        <h1>CSS文本阴影</h1>
        <p><b>text-shadow </b>属性为文本添加阴影</p>
        <p>text-shadow: 垂直阴影 水平阴影 模糊效果 颜色</p>
        <h2>文本阴影效果</h2>
    </body>
    </html>
