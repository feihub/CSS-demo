1. 文本颜色

color属性用于设置文本的颜色，颜色由以下值确定:

  • 颜色名 - 如color: green
  • 十六进制 - 如color: #ff0000
  • RGB值 - 如 color: rgb(255,0,0)

2. 文本背景色

backgroud-color属性用于设置背景颜色，背景颜色值的设定与文本颜色一样:

  • 颜色名 - 如 background-color: green
  • 十六进制 - 如 background-color: #aaa
  • RGB值 - 如 background-color: rgb(216, 127, 43)
