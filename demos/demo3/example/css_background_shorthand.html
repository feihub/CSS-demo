background 属性

background 属性是在一条声明中指定所有背景属性的简写属性。

在这里，背景图像只显示一次，并位于右上角。

我们还添加了右外边距，以使文本不会覆盖背景图片。

Code: <html> <head> <style> body { background: #696969 url("../assets/tree.png") no-repeat right top; } </style> </head> <body> <h1>background 属性</h1> <p>background 属性是在一条声明中指定所有背景属性的简写属性。</p> <p>在这里，背景图像只显示一次，并位于右上角。</p> <p>我们还添加了右外边距，以使文本不会覆盖背景图片。</p> </body> </html>
