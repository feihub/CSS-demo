outline-color 属性

outline-color 属性用于设置轮廓的颜色。

红色的实线轮廓。

蓝色的点状轮廓。

灰色的实线轮廓。

也可以使用十六进制值（HEX）, RGB值, HSL值指定轮廓颜色

Code: 
                <html>
                <head>
                    <style>
                        p.ex1 {
                        border: 2px solid black;
                        outline-style: solid;
                        outline-color: red;
                        }

                        p.ex2 {
                        border: 2px solid black;
                        outline-style: dotted;
                        outline-color: blue;
                        }

                        p.ex3 {
                        border: 2px solid black;
                        outline-style: outset;
                        outline-color: grey;
                        }
                    </style>
                </head>
                <body>

                <h1>outline-color 属性</h1>
                <p>outline-color 属性用于设置轮廓的颜色。</p>
