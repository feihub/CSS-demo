设置元素的高度和宽度

这个 div 元素的高度为 200 像素，宽度为 300px：

Code: 
            <html>
            <head>
              <style>
                .div-w-h {
                    height: 200px;
                    width: 300px;
                    background-color: powderblue;
                    margin: 30px;
                    padding: 50px;
                }
                .div-1 {
                    height: 200px;
                    width: 300px;
                    background-color: red;
        
                }
              </style>
            </head>
            <body>
        
              <h1>设置元素的高度和宽度</h1>
              <p>这个 div 元素的高度为 200 像素，宽度为 300px：</p>
              <div class="div-1"></div>
              <div class="div-w-h"></div>
        
            </body>
          </html>
