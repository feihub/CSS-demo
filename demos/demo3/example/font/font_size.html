CSS字体大小

font-size 属性用于设置字体的大小

1. 使用绝对长度单位px来设置字体大小

2. 使用相对长度单位em来设置字体大小

3. 响应式字体大小（请调整浏览器窗口大小，以查看字体变化）

使用vh 单位， 字体大小会根据浏览器窗口的高度变化而变大或变小

使用vw 单位， 字体大小会根据浏览器窗口的宽度变化而变大或变小

Code:

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>CSS字体大小</title>
            <style>
                .div_line {
                    width: 45%;
                    float: left;
                }
                .absolute_size {
                    font-size: 50px;
                }
                .relative_size {
                    font-size: 2em;
                }
                .p1 {
                    font-size: 3vh;
                }
                .p2 {
                    font-size: 2vw;
                }
            </style>
        </head>
        <body>
            <div class="div_line">
                <h1>CSS字体大小</h1>
                <p><b>font-size </b>属性用于设置字体的大小</p>
                <p class="absolute_size">1. 使用绝对长度单位px来设置字体大小</p>
                <p class="relative_size">2. 使用相对长度单位em来设置字体大小</p>
                <div class="response_size" style="border: 1px solid black;">
                    <p style="font-size: 4vw;">3. 响应式字体大小（请调整浏览器窗口大小，以查看字体变化）</p>
                    <p class="p1">使用vh 单位， 字体大小会根据浏览器窗口的高度变化而变大或变小</p>
                    <p class="p2">使用vw 单位， 字体大小会根据浏览器窗口的宽度变化而变大或变小</p>
                </div>
            </div>
        </body>
        </html>
