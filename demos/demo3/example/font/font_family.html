CSS通用字体族

在 CSS 中，我们使用 font-family 属性规定文本的字体。

在CSS中，有以下五个通用字体族:

通用字体族 样式
Serif Times New Roman
Georgia
Garamond
Sans-serif Arial
Verdana
Helvetica
Monospace Courier New
Lucida Console
Monaco
Cursive Brush Script MT
Lucida Handwriting
Fantasy Copperplate
Papyrus
