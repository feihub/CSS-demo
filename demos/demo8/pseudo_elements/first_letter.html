这是一个段落，我们将使用::first-letter伪元素来设置它的第一个字母的样式



Code: 
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>伪类:first-letter</title>
        <style>
            p::first-letter {
                font-size: 30px;
                background-color: salmon;
            }
        </style>
    </head>
    <body>
        <div class="content">
            <p>这是一个段落，我们将使用::first-letter伪元素来设置它的第一个字母的样式</p>
        </div>
    </body>
    </html>
