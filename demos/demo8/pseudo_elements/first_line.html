CSS是Cascading Style Sheets（层叠样式表单）的缩写，它是一种用来表现HTML或XML 等文件式样的计算机语言。 CSS的作用就是为网页设置外观，相当于给HTML文档穿上了华丽的衣服。 CSS目前最新版本为CSS3，能够真正做到网页表现与内容分离的一种样式设计语言。



Code: 
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>伪元素::first-line</title>
        <style>
            p::first-line {
                font-size: 26px;
                color: red;
            }
        </style>
    </head>
    <body>
