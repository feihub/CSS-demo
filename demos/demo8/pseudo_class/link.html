:link选择未被访问的链接，并设置其样式：

Google


Code: 
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>伪类:link</title>
        <style>
            a:link {
                background-color: cyan;
            }
        </style>
    </head>
    <body>
        <div class="content">
            <p>:link选择未被访问的链接，并设置其样式：</p>
            <a href="http://www.google.cn/" target="_blank">Google</a>
        </div>
    </body>
    </html>
