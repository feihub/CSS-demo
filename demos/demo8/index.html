1. 伪类

  • 定义：

    • 伪类用于定义元素的特殊状态

  • 语法：
    • selector:pseudo-class {
        property: value;
    }

常用的伪类有：

选择器 例子 描述
:active a:active 向被激活的元素添加样式
:focus input:focus 选择获得焦点的input元素。
:hover a:hover 当鼠标悬停在a元素上方时，向元素添加样式
:link a:link 向未访问的链接添加样式
:fisrt-child p:first-child 选择属于其父元素的首个子元素的每个p元素，并为其设置样式

2. 伪元素

  • 定义：

    • 伪元素用于设置元素指定部分的样式

  • 语法：
    • selector::pseudo-element {
        property: value;
    }

伪元素有以下几种：

选择器 例子 描述
::first-letter p::first-letter 选择每个p元素的首字母
::first-line p::first-line 选择每个p元素的首行
::after p::after 在每个P元素之后添加内容
::before p:before 在每个p元素之前添加内容
