[target=_blank]:选择带有target = "_blank" 属性的元素

1. 带有target属性，但是为空的a元素

2. 带有target属性且值为_blank的a元素

3. 没有带target的a元素

Code:

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>[target=_blank]</title>
        <style>
            /* 选择带有target = "_blank" 属性的元素 并将其字体颜色设置为橙色 */
            a[target="_blank"] {
                color: orange;
            }
        </style>
    </head>
    <body>
        <h3>[target=_blank]:选择带有target = "_blank" 属性的元素</h3>
        <p><a href="" target="">1. 带有target属性，但是为空的a元素</a></p>
        <p><a href="" target="_blank">2. 带有target属性且值为_blank的a元素</a></p>
        <p><a href="">3. 没有带target的a元素</a></p>
    </body>
    </html>
