[lang|en]: 选择lang属性值以"en"开头的元素

1. 带有lang属性且值以"en"开头的p元素

2. 带有lang属性且值以"en"开头的a元素

3. 带有lang属性且值以"zh"开头的p元素

Code:

    <!DOCTYPE html>
    <html>
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>[attribute|value]</title>
        <style>
            /* 选择lang属性值以"en"开头的元素并将背景颜色设置为黄色 */
            [lang|=en] {
                background:yellow;
            }
        </style>
    </head>
    <body>
        <h3>[lang|en]: 选择lang属性值以"en"开头的元素</h3>
        <p lang="en">1. 带有lang属性且值以"en"开头的p元素</p>
        <a lang="en-us">2. 带有lang属性且值以"en"开头的a元素</a>
        <p lang="zh">3. 带有lang属性且值以"zh"开头的p元素</p>
    </body>
    </html>
