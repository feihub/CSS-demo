[target]：选择带有target属性的所有元素

1. 带有target值为空的a元素

2. 带有target值为_blank的a元素

3. 没有带target的a元素

Code

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>[attribute]</title>
        <style>
            /* 将带有target属性的元素字体设置为红色 */
            a[target] {
                color: red;
            }
        </style>
    </head>
    <body>
        <h3>[target]：选择带有target属性的所有元素并将字体设置为红色</h3>
        <p><a href="" target="">1. 带有target值为空的a元素</a></p>
        <p><a href="" target="_blank">2. 带有target值为_blank的a元素</a></p>
        <p><a href="">3. 没有带target的a元素</a></p>
    </body>
    </html>
