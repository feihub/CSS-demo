1. 带计数器的自动编号

CSS计数器就像“变量”。变量值可以通过 CSS 规则递增（将跟踪它们的使用次数）。

在使用CSS计数器之前，必须先使用counter-reset来创建它

HTML教程

Javascript教程

Code:

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>计数器</title>
        <style>
            /* 首先使用counter-reset在body选择器中创建一个计数器 */
            body {
                counter-reset: course;
            }
            /* 在h2的前面插入内容 */
            h2::before {
                /* 递增多个course计数器 */
                counter-increment: course;
                /* counter(计数器名) */
                content: "Course" counter(course) ": ";
            }
        </style>
    </head>
    <body>
        <div class="div_line">
            <h1>1. 带计数器的自动编号</h1>
            <p><b>CSS计数器就像“变量”。变量值可以通过 CSS 规则递增（将跟踪它们的使用次数）。</b></p>
            <p>在使用CSS计数器之前，必须先使用<b>counter-reset</b>来创建它</p>
            <h2>CSS教程</h2>
            <h2>HTML教程</h2>
            <h2>Javascript教程</h2>
        </div>
    </body>
    </html>
