div1: 宽度=500px, 高度=200px


div2: 宽度=500px, 高度=200px

使用box-sizing属性将内边距和边框包含在元素的总宽度和高度中


div3: 宽度=500px, 高度=200px

div3尺寸不同于div1是因为它设置了内边距

Code:

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Box-Sizing</title>
            <style>
                .box1 {
                    width: 500px;
                    height: 200px;
                    border: 1px solid black;
                }
                /* box-sizing属性允许元素的总宽度和高度中包括内边距和边框 */
                .box2 {
                    width: 500px;
                    height: 200px;
                    padding: 50px;
                    border: 1px solid red;
                    box-sizing: border-box;
                }
                .box3 {
                    width: 500px;
                    height: 200px;
                    padding: 50px;
                    border: 1px solid green;
                }
            </style>
        </head>
        <body>
            <div class="box1">
                <p>div1: 宽度=500px, 高度=200px</p>
            </div>
            <br>
            <div class="box2">
                <p>div2: 宽度=500px, 高度=200px</p>
                <p>使用box-sizing属性将内边距和边框包含在元素的总宽度和高度中</p>
            </div>
            <br>
            <div class="box3">
                <p>div3: 宽度=500px, 高度=200px</p> 
                <p>div3尺寸不同于div1是因为它设置了内边距</p>
            </div>
        </body>
        </html>
