Code: 

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="https://cdn.staticfile.org/twitter-bootstrap/3.3.7/css/bootstrap.min.css">  
            <script src="https://cdn.staticfile.org/twitter-bootstrap/3.3.7/js/bootstrap.min.js"></script>
            <title>CSS网格布局</title>
            <style>
                body {
                    margin: 20px;
                }
                .grid_container {
                    display: grid;
                    /* grid-template-columns 属性定义网格布局中的列数，并可定义每列的宽度 */
                    grid-template-columns: auto auto auto;
                    background-color: peachpuff;
                    padding: 40px;
                    /* grid-column-gap属性设置每列的间隙 */
                    grid-column-gap: 20px;
                    /* grid-column-gap属性设置每列的间隙 */
                    grid-row-gap: 20px;
                }
                .grid_item {
                    background-color: #eee;
                    border: 1px solid black;
                    padding: 30px;
                    text-align: center;
                }
                /* 从第一列开始到第三列结束 */
                .item1 {
                    grid-column-start: 1;
                    grid-column-end: 3;
                }
                /* 从第二行开始到第四行结束 */
                .item6 {
                    grid-row-start: 2;
                    grid-row-end: 4;
                }
            </style>
        </head>
        <body>
            <h3>如需使用网格布局模型，可以将 HTML 元素的 display 属性设置为 grid 或者 inline-grid，它就会成为网格容器。</h3>

                <div class="grid_item">item4</div>
                <div class="grid_item">item5</div>

                <div class="grid_item">item7</div>
                <div class="grid_item">item8</div>
                <div class="grid_item">item9</div>
            </div>
