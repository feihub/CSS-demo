justify-content属性用于对齐 flex 项目

center 值将 flex 项目在容器中居中对齐

1

flex-start 值将 flex 项目在容器的开头对齐（默认）:

2

flex-end 值将 flex 项目在容器的末尾对齐:

3

Code:

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link rel="stylesheet" href="https://cdn.staticfile.org/twitter-bootstrap/3.3.7/css/bootstrap.min.css">  
        <script src="https://cdn.staticfile.org/twitter-bootstrap/3.3.7/js/bootstrap.min.js"></script>
        <title>justify content</title>
        <style>
            body {
                padding: 20px;
            }
            .flex {
                display: flex;
                flex-direction: row;
                background-color: #ccc;
                justify-content: center;
            }
            .flex > div {
                margin: 10px;
                padding: 20px;
                width: 50px;
                background-color: white;
            }
            /* 居中对齐 */
            .container1 {
                justify-content: center;
            }
            /* 向开头对齐 */
            .container2 {
                justify-content: flex-start;
            }
            /* 向末端对齐 */
            .container3 {
                justify-content: flex-end;
            }
        </style>
    </head>
    <body>
        <div class="div_line">
            <h1><code>justify-content</code>属性用于对齐 flex 项目</h1>
            <h3><code>center</code> 值将 flex 项目在容器中居中对齐</h3>
            <div class="flex container1">
                <div>1</div>
            </div>
            <h3><code>flex-start</code> 值将 flex 项目在容器的开头对齐（默认）:</h3>
            <div class="flex container2">
                <div>2</div>
            </div>
            <h3><code>flex-end</code> 值将 flex 项目在容器的末尾对齐（默认）:</h3>
            <div class="flex container3">
                <div>3</div>
            </div>
        </div>
    </body>
    </html>
