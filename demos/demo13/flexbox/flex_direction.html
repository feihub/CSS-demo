如需使用 Flexbox 模型，首先需定义flex容器

弹性布局中必须有一个 display 属性设置为 flex 的父元素

1
2
3

Code:

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>flex-direction</title>
        <style>
            .flex-container {
                /* 通过将 display 属性设置为 flex，flex 容器将可伸缩 */
                display: flex;
                background-color: #aaa;
                /* flex-direction: column; 垂直堆叠弹性项目（从上到下） */
                flex-direction: column;
            }
            /* > 子选择器 */
            .flex-container > div {
                margin: 20px;
                padding: 30px;
                width: 30px;
                background-color: white;
                text-align: center;
            }
        </style>
    </head>
    <body>
        <h3>如需使用 Flexbox 模型，首先需定义flex容器</h3>
        <p>弹性布局中必须有一个 display 属性设置为 flex 的父元素</p>
        <div class="flex-container">
            <div>1</div>
            <div>2</div>
            <div>3</div>
        </div>
    
    </body>
