用媒体查询隐藏元素

当屏幕尺寸小于600px时，隐藏下面的div

隐藏元素
            
Code:

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>隐藏元素</title>
            <style>
                body {
                    margin: 0;
                }
                .hide_element {
                    padding: 30px;
                    background-color: #aaa;
                }
                /* 当屏幕尺寸小于600px时，隐藏div */
                @media only screen and (max-width: 600px) {
                    .hide_element {
                        display: none;
                    }
                }
            </style>
        </head>
        <body>
            <h1>用媒体查询隐藏元素</h1>
            <p>当屏幕尺寸小于600px时，隐藏下面的div</p>
            <div class="hide_element">隐藏元素</div>
        </body>
        </html>
